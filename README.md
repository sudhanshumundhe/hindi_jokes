# 😂 JokeHub App

A simple and fun **React web application** that displays random jokes in a clean and modern interface. The app provides an enjoyable experience with smooth interactions and a minimal design.

---

## 🌐 Live Demo

👉 https://hindijoke.netlify.app/

---

## ✨ Features

* Clean and centered UI
* Random jokes display
* Next joke button
* Responsive design
* Smooth user experience

---

## 🧑‍💻 Languages & Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Node.js (for development)

---

## 🚀 How to Run

### 1️⃣ Clone the project

```bash id="run1"
git clone https://github.com/sudhanshumundhe/hindi_jokes.git
```

### 2️⃣ Move into project folder

```bash id="run2"
cd hindi_jokes
```

### 3️⃣ Install dependencies

```bash id="run3"
npm install
```

### 4️⃣ Start development server

```bash id="run4"
npm run dev
```
